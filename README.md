# ExamGift-Android
梦幻西游科举答题神器

-------

Features/功能点
-------
* 支持中文、拼音快速检索答案
* 涵盖99%的题库练习  
* 提供错题练习，提升正确率
* 自定义主题色

核心技术点
-------
* 题库加密,利用nio对文件通道二进制数据进行打乱,破坏文件结构
